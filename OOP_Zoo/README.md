The objective of this project was to learn the basic concepts of OOP in PHP by recreating a zoo

> Code to be run via the app.php file

# Features

- Animal Class
- Different animals to put in the zoo
- Enclosures
  - Aviary
  - Fence
  - Aquarium
- Interfaces
  - CanWalk
  - CanFly
  - CanSwim
